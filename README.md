# Endava DEV Week challenge 2018

## Build
```
git clone https://github.com/Ernyoke/join-the-code.git
mvn clean install
```

### Build for Google cloud
```
git clone https://github.com/Ernyoke/join-the-code.git
git checkout google-cloud
mvn -DskipTests appengine:deploy
```
